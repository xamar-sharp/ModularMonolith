# ModularMonolith

## Описание
Часть единого модульного монолита, отвечающая за входную точку системы и обработку вывода остальных модулей.
## Установка
Выполните эту команду в командной строке Windows:
```bash
	git clone https://github.com/xamar-sharp/ModularMonolith
```

## Использование
Опубликовано в development-форме и может быть вами модифицировано,
при наличии установленных на вашем устройстве платформы `.NET9`.
